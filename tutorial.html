User Interface

The user interface has four key components that each have a unique role.

  • 1. Main Viewport

    This is the main window that displays everything that is being rendered onto to screen and is the component which users will use to interact with the virtual environment.

  • 2. Environment Panel

  • 3. Object Editor Panel

  • 4. Console

    The console window display all internal log messages onto the screen. Messages are categorized into three levels.

    • Info

      Messages that are considered informational only.

    • Warning

      Messages that indicate an event of significance has occurred but is not critical.

    • Error

      This is the most critical message type and indicates that something severe has occurred which resulted in a failure.

Controls

Shortcuts can also be accessed within application by going to the main menu and clicking on Help > Shortcuts.

Action Shortcut
Load Model Ctrl + L
Export Model Ctrl + E
Toggle Viewport F1
Toggle Fullscreen F2

Importing models

Importing models is one of the fundamental aspects of VMVE and can be achieved by going onto the main menu and clicking on File > Load Model as seen in the image below.

Once clicked, a new window will appear that will allow you to navigate your filesystem and load a model file.

Exporting models

VMVE also has the ability to export models into its own custom file format. The export window can be access by going to File > Export model as seen in the image below.

Once clicked, a new window will appear that will allow you to navigate your filesystem and export a model.
