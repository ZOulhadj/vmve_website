VMVE

A lightweight 3D environment visualizer.

Find out more Download

Features

  • Lightweight (Only 4mb)
  • Vulkan rendering API
  • Supports 64bit world positions
  • Deferred rendering pipeline
  • G-Buffer visualization
  • Asynchronous model loading
  • Quaternion camera system
  • Model encryption
  • Dynamic virtual environment
  • Custom vmve file format
  • Model format encryption
  • Powered by Custom Engine
